High-Speed Server Log Analyzer (Task 2)
Description

This project processes large server log files efficiently using Java. It reads thousands of log entries and extracts useful information such as errors and user activity.

Tech Stack

Java
Java NIO.2
Regular Expressions
Streams API

Features

Fast file reading using NIO.2
Pattern matching using Regular Expressions
Stream-based data processing
Handles large datasets efficiently

Files

LogAnalyzer.java – Main program
log.txt – Sample input log file

How to Run

Compile:

javac LogAnalyzer.java

Run:

java LogAnalyzer
Output

Displays processed log data
Extracts useful information like errors and activities

Learning Outcome

Learned how to handle large files, optimize performance, and process log data efficiently using Java.
