# Multi-Threaded Event Ticketing System (Task 3)

## Description
This project simulates a high-demand ticket booking system where multiple users try to book limited tickets at the same time.

## Tech Stack
Java  
Thread API  
ExecutorService  
ReentrantLock  

## Features
Handles multiple users using threads  
Prevents race conditions  
Avoids double booking using locks  
Efficient thread management  

## How to Run

Compile:
javac TicketBookingSystem.java

Run:
java TicketBookingSystem

## Output
Shows booking attempts and successful ticket allocations

## Learning Outcome
Learned multithreading, synchronization, and concurrency handling in Java.
